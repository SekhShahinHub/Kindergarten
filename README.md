# 🌈 Little Learner's – Kindergarten Website

A modern, responsive, and child-friendly kindergarten website designed to provide parents with an engaging and informative experience. The website features a clean UI, colorful design, smooth animations, and a fully responsive layout for desktops, tablets, and mobile devices.

## ✨ Features

- 🎨 Modern and minimal UI design
- 📱 Fully responsive layout
- ⚡ Smooth animations and transitions
- 👶 Child-friendly color palette
- 🏫 Hero section with call-to-action
- 📚 Programs/Courses section
- 👩‍🏫 Teachers section
- 🖼️ Gallery
- 💬 Testimonials
- 📅 Admissions section
- 📞 Contact section
- 🌐 Cross-browser compatible

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Responsive Web Design
- CSS Animations

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/yourusername/little-learners.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.

No installation or build tools are required.

## 🎯 Purpose

This project was created as a modern frontend website for a kindergarten or preschool. It focuses on creating a warm, welcoming experience for parents while showcasing educational programs and school information.
